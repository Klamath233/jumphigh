JumpHigh
===
Predict how high you will jump.

# Dependencies
### macOS
First of all, you need Homebrew.  
Install python3:  
`brew install python3`  
Install dependencies:  
`pip3 install openpyxl scipy sklearn matplotlib`  

### Ubuntu/Debian
Install python3:  
`apt install python3`  
Install dependencies:  
`pip3 install openpyxl scipy sklearn matplotlib`  

# Run
`python3 ./src/maching_learning.py`
